#Engenharia de Dados Destaque

Sistema Automatizado de Análise de Grade para Ingressantes

Desenvolvimento de uma solução inteligente para análise e mapeamento de matrizes curriculares, voltada para a dispensa de matérias de alunos ingressantes na faculdade. O sistema otimiza o fluxo da secretaria acadêmica, processando equivalências de forma ágil e segura.

Destaques do Projeto:

  • Cruzamento de dados estruturados entre diferentes grades horárias e ementas.
  • Redução drástica no tempo de resposta para a efetivação da matrícula com aproveitamento de estudos.
Resultado Real: Automação de processos burocráticos internos e melhoria na experiência de onboarding do novo acadêmico.

SQL Server n8n ETL Sistemas Internos
Status: ● Ativo em Produção

#IA & Conversacional Automação

Validador de Leads & Chatbot Corporativo via RD Conversas

Desenvolvimento de uma camada de higienização de dados que valida e-mails e telefones em tempo real, integrada a um fluxo conversacional construído no RD Conversas. Além do fluxo padrão, foi implementado com sucesso um MVP utilizando a arquitetura LangChain via API REST para atendimento inteligente e contextualizado.

Arquitetura do Fluxo:

  • Filtro Regex e APIs externas para barrar contatos inválidos antes de entrarem no funil de vendas.
  • Integração REST bem-sucedida injetando o poder do LangChain diretamente dentro das automações do RD Conversas.
Resultado Real: Base de contatos limpa, redução de custos com mensagens perdidas e validação técnica de IA generativa em canais de chat.
Tecnologias Utilizadas
RD Conversas LangChain API REST JavaScript
Status: ● MVP / Produção
